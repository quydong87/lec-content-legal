Privacy Policy

Last updated: June 5, 2025  ·  Effective immediately

1. Overview

LeC Content ("we", "our", "the Service") is an internal content automation tool developed by LeC Studio. This Privacy Policy explains how we collect, use, and protect information when you authorize our application to interact with your TikTok account via the official TikTok Content Posting API.

We are committed to protecting your privacy and handling your data responsibly.

2. Information We Collect

Data TypeWhat We CollectWhy
OAuth Access Token TikTok access token and refresh token To authenticate API requests and publish video content on your behalf
User Identifier TikTok Open ID (anonymous identifier) To associate tokens with the correct TikTok account
Basic Profile Info Display name, avatar URL (from TikTok API) To confirm correct account identity during setup
Video Metadata Video title, caption, privacy setting you provide To correctly publish content via TikTok API

We do not collect passwords, payment information, or any sensitive personal data.

3. How We Use Your Information

All collected data is used solely to:

• Authenticate with TikTok API on your behalf
• Publish video content to your authorized TikTok account(s)
• Refresh expired tokens automatically to maintain uninterrupted service
• Send notifications about publishing status via Telegram (internal team only)

We do not sell, trade, or share your data with any third parties outside of TikTok's official API infrastructure.

4. Data Storage

OAuth tokens are stored locally on the authorized device in an encrypted JSON file (assets/.tiktok/tokens.json). Tokens are never transmitted to any external server operated by LeC Studio.

The only external communication is directly with TikTok's official API servers (open.tiktokapis.com).

5. Data Retention

Token data is retained only as long as the Service is in active use. You can delete all stored tokens at any time by removing the assets/.tiktok/tokens.json file from your device.

6. Your Rights & Revoking Access

You can revoke this application's access to your TikTok account at any time:

• Open TikTok app → Profile → Settings and privacy
→ Security → Manage app permissions
→ Find "LeC Content" → Revoke access

Upon revocation, all stored tokens become invalid immediately.

7. Third-Party Services

This application integrates with:

• TikTok Content Posting API — subject to TikTok's Privacy Policy
• Facebook Graph API — for source content scanning, subject to Meta's Privacy Policy
• Google Gemini API — for AI caption generation (text only, no personal data)

8. Security

All API communications use HTTPS encryption. Tokens are stored locally and never transmitted to third-party servers. Access to the tool is restricted to authorized LeC Studio personnel only.

9. Children's Privacy

This Service is not directed to children under 13. We do not knowingly collect personal information from children.

10. Changes to This Policy

We may update this Privacy Policy from time to time. The "Last updated" date at the top will reflect any changes. Continued use of the Service constitutes acceptance of the updated policy.
